# Code of Conduct

## Our standard

This project expects respectful, constructive collaboration.

Examples of behavior that help the project:

- giving actionable technical feedback
- assuming good intent
- keeping disagreement focused on code, design, and evidence
- welcoming contributors with different backgrounds and experience levels

Examples of unacceptable behavior:

- harassment, insults, or personal attacks
- discriminatory or sexualized language
- repeated bad-faith arguments or deliberate disruption
- publishing private information without consent

## Scope

This applies to project spaces, including:

- GitHub issues and pull requests
- discussions and code review
- other public communication channels used for this project

## Enforcement

Project maintainers may remove comments, close threads, or restrict participation when behavior harms the project or its contributors.

## Reporting

If you experience or observe unacceptable behavior, open a private security-style report through GitHub or contact the maintainer directly through the repository owner profile.
